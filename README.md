# Курсовая работа: Denoising Автоэнкодеры (полносвязный и сверточный)

## Описание проекта

Курсовая работа по нейронным сетям, реализующая два варианта denoising автоэнкодера для восстановления зашумленных изображений:
1. **Сверточный denoising автоэнкодер**
2. **Полносвязный denoising автоэнкодер**

Работа выполнена на датасете MNIST и соблюдает ограничение на количество параметров (≤ 5 млн).

## Структура проекта

```
.
├── README.md                 # Инструкция по запуску
├── requirements.txt          # Зависимости проекта
├── denoising_ae.py          # Модуль denoising автоэнкодера (conv и FC)
├── utils.py                 # Вспомогательные функции
├── experiments.ipynb        # Ноутбук с экспериментами
└── config.py                # Конфигурация экспериментов
```

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите ноутбук `experiments.ipynb`


### Быстрый старт

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Откройте ноутбук:
```bash
jupyter notebook experiments.ipynb
```

3. Запустите ячейки последовательно

### Использование модулей отдельно

Все модули можно использовать отдельно, см. примеры в ноутбуке.

## Параметры моделей

- **ConvDenoisingAE**: ~1.8M параметров
- **FCDenoisingAE**: ~1.2M параметров

Обе модели соответствуют ограничению на количество параметров (≤ 5 млн).

## Воспроизводимость

Все эксперименты используют фиксированный seed (42) для воспроизводимости результатов.
